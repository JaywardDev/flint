# Flint

Flint V1 Core is a simple personal notebook for short history records.

V1 Core is functionally complete: Add, List, Detail, and one Record shape. The current doctrine now permits only lightweight post-V1 secondary views, such as `/timeline` and `/sparks`, when they remain lenses over existing records and do not change capture.
